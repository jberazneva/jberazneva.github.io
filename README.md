# juliaberazneva.com

Source for [juliaberazneva.com](https://juliaberazneva.com) — 
academic website of Julia Berazneva, Associate Professor of Economics, 
Middlebury College.

## Updating the site

All content lives in `content-source/`. Edit a file there, commit, and 
push — GitHub Actions rebuilds and redeploys automatically.

**Add a publication** → edit `content-source/publications.yml`  
**Add a working paper** → edit `content-source/working-papers.yml`  
**Update bio** → edit `content-source/bio.md`  
**Update CV** → drop new PDF into `reference/` as `Berazneva_CV-MonthYear.pdf`  
**Update contact info or affiliations** → edit `content-source/contact.yml`
